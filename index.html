This app has two parts: a Python backend that scrapes Instagram and calls AI, and this HTML frontend you're looking at now. Follow the steps below to get everything running in about 5 minutes.

Step 01
Clone the backend repo
Run in your terminal:
git clone https://github.com/sakshith899-afk/viral-reel-scraper
cd viral-reel-scraper
Step 02
Create a virtual environment
python -m venv venv
venv\Scripts\activate (Windows)
source venv/bin/activate (Mac/Linux)
Step 03
Install dependencies
pip install -r requirements.txt

This installs FastAPI, OpenAI SDK, Apify client, and all other dependencies.
Step 04
Set up your API keys
Copy .env.example to .env and fill in your tokens. See the token guide below for where to get each one.
Step 05
Start the backend server
uvicorn api:app --reload

You should see: Uvicorn running on http://127.0.0.1:8000
Step 06
Open this file & generate ideas
Open this HTML file in your browser. Type your niche (e.g. "fitness routines") and click Generate Ideas. Results appear in ~30–60 seconds.
API Tokens You Need
GITHUB_TOKEN Free AI (recommended). Go to GitHub → Settings → Developer settings → Personal access tokens → Fine-grained → Generate new token. No special permissions needed. Gives you free access to GPT-4o via GitHub Models.
APIFY_API_TOKEN For real Instagram scraping. Sign up at apify.com → Settings → Integrations → API token. Free tier gives ~$5 credit/month. Without this, the app runs on mock data (still useful for testing).
OPENAI_API_KEY Optional — only if not using GITHUB_TOKEN. platform.openai.com → API Keys → Create new. Costs money per call. Skip this if you have GITHUB_TOKEN set.
